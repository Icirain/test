Detailed Description

The main class for the simulation of a 2D world with many robots running around.

RobotViewer or Tests call set_time and advance_time to control the simulation and use the get*() functions to read out the current state of the simulation (i.e., the current positions and orientations of robots and obstacles).

For now, RobotLand is hard coded to run a simulation of two robots running around in a circle. You can see their sensors, but they don't yet respond to each other or to obstacles.
